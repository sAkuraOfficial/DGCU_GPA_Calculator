###### 开发前的配置要求

1. Visual Studio 2022 (可选MinGW + VS Code)
2. QT6
3. CMake
4. VCPKG
5. Git

###### 编译项目

1. 克隆项目
2. 安装仓库自带的VCPKG清单
3. 打开Visual Studio 2022, 点击`打开文件夹`, 打开本项目
4. 点击`生成`->`全部生成`即可完成编译

```sh
git clone https://github.com/sAkuraOfficial/DGCU_GPA_Calculator.git
cd ./DGCU_GPA_Calculator
vcpkg install
```
